# 车间风险四色图系统

## 快速开始

### 1️⃣ 打开后台管理页面
双击打开 `workshop-risk-admin.html`

### 2️⃣ 打开展示页面
双击打开 `workshop-risk-display.html`

### 3️⃣ 配置数据
在后台管理页面：
- 📤 上传车间平面图
- ✏️ 绘制区域范围
- ⚠️ 添加风险点

### 4️⃣ 查看展示
在展示页面点击"刷新数据"按钮即可查看最新数据

## 文件说明

| 文件名 | 说明 | 是否必需 |
|--------|------|----------|
| workshop-risk-admin.html | 后台管理页面 | ✅ 必需 |
| workshop-risk-display.html | 展示页面 | ✅ 必需 |
| xlsx.full.min.js | Excel处理库 | ✅ 必需 |
| 使用说明.txt | 详细使用说明 | 📖 建议阅读 |
| README.md | 快速入门文档 | 📖 本文件 |

## 主要功能

### 后台管理
- 🖼️ 上传车间平面图
- 📦 绘制和管理区域
- ⚠️ 添加和编辑风险点
- 🎨 设置风险等级（红/橙/黄/蓝）
- 📊 批量导入Excel数据
- 🖼️ 上传区域四色图

### 展示页面
- 📍 显示所有风险点
- 📊 实时统计风险数量
- 🔍 查看风险点详情
- 🖼️ 查看区域四色图
- 🔄 自动/手动刷新数据

## 风险等级

- 🔴 **红色** - 重大风险
- 🟠 **橙色** - 较大风险
- 🟡 **黄色** - 一般风险
- 🔵 **蓝色** - 低风险

## 注意事项

⚠️ **重要提示**
- 两个页面需要在同一个浏览器中打开
- 数据保存在浏览器本地（localStorage）
- 建议定期导出数据备份
- 支持Chrome、Edge、Firefox等现代浏览器

## 跨电脑使用

将整个文件夹复制到其他电脑即可使用，无需安装任何软件。
